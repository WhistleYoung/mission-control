# Mission Control 更新日志

## v1.0.8 (2026-04-02)

### ✨ 功能优化
- **实时会话页面优化**：
  - 布局改为左右排版：左侧「后台运行任务」，右侧「实时会话」
  - 新增杀死空闲子进程功能
  - 任务类型标记：「Subagent」(紫色) / 「主进程」(蓝色)
  - 状态指示：绿色脉冲(运行中) / 灰色(空闲) / 红色(失败)
- **系统会话过滤**：自动过滤 cron、heartbeat、openai 系统会话，只显示真正的对话类会话
- **模型编辑功能**：支持编辑模型参数（上下文窗口、最大输出、输入类型、推理模型）
- **模型删除保护**：删除模型前检查是否被 agent 绑定，如有绑定则提示

### 🐛 问题修复
- 修复员工模块模型显示 bug（openclaw.json 优先级问题）
- 修复删除模型时 agents.defaults.models 引用未清理问题
- 修复杀死子进程失败问题（改用 sessions.delete 替代 sessions.abort）

### 🎨 界面优化
- 实时会话页面支持拖拽滚动
- 模型弹窗支持滚动，防止内容过多遮挡
- 版本号更新为 v1.0.8

## v1.0.7 (2026-04-02)

### ✨ 功能优化
- **模型模块全面优化**：
  - 新增 Anthropic 接口协议支持（与 OpenAI 兼容协议切换）
  - 新增批量添加模型功能：一次可添加多个模型（一行一个 `id:显示名称`）

### 🐛 问题修复

### 🎨 界面优化
- 模型列表弹窗支持滚动，防止内容过多遮挡
- 批量添加模型使用 textarea，支持多行输入和自动解析
- 版本号更新为 v1.0.7

## v1.0.5 (2026-04-01)

### 🐛 问题修复
- 修复龙虾办公室无法隐藏的问题（settings API 的 MySQL 语法与 SQLite 不兼容）

### ✨ 功能优化
- **数据库改造**：从 MySQL 迁移到 SQLite（better-sqlite3），零依赖、开源友好
  - 自动初始化数据库和默认管理员账号（admin/admin123）
  - 数据库文件存储在 `data/mission-control.db`，自动创建 `data/` 目录
- **技能模块全面优化**：
  - 新增「搜索安装」功能，对接 ClawHub.ai 技能市场
  - 已安装技能按 Agent 分组展示，清晰显示每个技能属于哪个 Agent
  - 技能名称中文映射（如 `paddleocr-doc-parsing` → `文档图片OCR解析`）
  - 点击技能卡片弹出详情弹窗，显示完整描述
  - 新增「复制技能」功能：可将技能一键复制到其他指定 Agent
  - 新增「删除技能」功能：可从指定 Agent 删除技能

### 🎨 界面优化
- 日志记录按倒序排列，最新记录显示在最前面
- 技能详情弹窗新增目标 Agent 选择器，支持多选复制
- 版本号更新为 v1.0.5
